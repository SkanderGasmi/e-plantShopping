# e-plantShopping - Paradise Nursery

## Project Overview
Paradise Nursery is a React-based shopping application for houseplants. This application allows users to browse various houseplants categorized by type, add them to a shopping cart, and manage cart items before checkout.

## Features
- Three main pages: Landing, Product Listing, and Shopping Cart
- Product categorization (Air Purifying, Aromatic, Medicinal, etc.)
- Redux-based state management for cart functionality
- Dynamic cart updates with quantity adjustment
- Responsive design

## Tech Stack
- React
- Redux Toolkit
- React Router
- CSS

## Getting Started
1. Clone the repository
2. Run `npm install`
3. Run `npm start`
